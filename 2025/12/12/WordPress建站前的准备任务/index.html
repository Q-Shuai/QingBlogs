WordPress建站前的准备任务

Qing2025-12-122026-07-19

WordPress 建站 · 准备任务

任务一：准备网络环境（梯子）

重要：后续所有实操都需要稳定的梯子，请提前准备好。

任务二：配置谷歌浏览器与谷歌账号

本课程全程基于谷歌浏览器操作，请提前完成以下配置。

2.1 下载并安装谷歌浏览器

下载地址：https://www.google.com/intl/zh-CN/chrome/

如果打不开页面，说明未开梯子，打开梯子后可正常下载。

2.2 将谷歌浏览器设为默认浏览器

Windows 电脑：

  1. 打开「设置」→「应用」→「默认应用」
  2. 在「Web 浏览器」下选择「Google Chrome」

Win7 系统无法下载最新版 Chrome，可跳过此步骤。

Mac 电脑：

  1. 打开「系统设置」→「桌面与坞」
  2. 找到「默认 Web 浏览器」，选择「Google Chrome」

2.3 检查浏览器版本

  1. 打开谷歌浏览器
  2. 点击右上角「三点」菜单 →「帮助」→「关于 Google Chrome」
  3. 确保已更新到最新版本

2.4 关联桌面版微信（可选）

这样在微信里点开链接直接用 Chrome，而非微信自带的受限浏览器。

  1. 打开微信「设置」
  2. 在「通用设置」中勾选「使用系统默认浏览器打开链接」

2.5 注册谷歌账号（Gmail）

已有账号可跳过此步骤。

  • 手机开启梯子，访问 https://accounts.google.com/
  • 用途选择个人用途，所有信息如实填写
  • 手机号填 +86 国内手机号
  • 超过 30 分钟未成功则放弃，直接看后续教程
  • 备选：在 YouTube 搜索「怎么注册 gmail 账号」找最新教程

任务三：安装浏览器插件

3.1 沉浸式翻译插件

强大的网页翻译工具，对外贸工作也大有裨益。

下载地址：https://chromewebstore.google.com/detail/bpoadfkcbjbfhfodiogcnhhhpibjhbnh

安装后建议在扩展栏中置顶。

3.2 长截图工具（Chrome Capture）

下载地址：https://chromewebstore.google.com/detail/chrome-capture-screenshot/ggaabchcecdbomdcnbahdfddfikjmphe

任务四：购买域名

域名按年续费，并非一次性买断，到期不续费将进入失效流程。

建议使用新域名实操；如使用已有域名，请确保无旧内容。

4.1 选择注册平台

建议自己注册域名——域名代注册等于买房却写别人名字。

条件 推荐平台
有双币信用卡或 PayPal Cloudflare
仅有支付宝 Namesilo（支持支付宝、信用卡、PayPal）

4.2 购买教程

  • Cloudflare：查看教程
  • Namesilo：查看教程

任务五：购买空间

空间（包括域名、网站虚拟主机、云服务器等）的本质是租赁服务，用户通过支付年度费用获得一定期限内的使用权，而非永久所有权。到期后若需继续使用，必须重新缴纳费用，否则空间将被回收。

5.1 怎么购买空间

空间可以买时间长一点（四年），因为后面续费的话，一年是现在差不多四年的费用

参考教程： https://dubble.so/guides/hostinger-799-jhcznolu7lrtzw5l8czv

5.2 怎么在新空间创建网站？

  • 切换网站为中文
  • 购买后的设置
    • 参考教程：https://dubble.so/guides/-dng4y5i54anlqcewr8ny

5.3 关联 Cloudfare/阿里云/腾讯云/新网/godday 域名教程

通过 A 记录关联阿里云、腾讯云、Cloudfare、新网、godady购买的域名：

阿里云、腾讯云、新网解析前确保账号已经通过实名认证

参考教程：https://dubble.so/guides/hostinger-cloudfaregodady-kj5psci2dlu4bcusodc5

任务六：生成 AI API 密钥

为后续课程调用 AI 功能做准备。

6.1 注册 AIHubmix 并充值

  1. 访问 AIHubmix 注册账号
  2. 进入「在线充值」，选择 Alipay+ 扫码支付

6.2 创建 API 密钥

  1. 进入 AIHubmix 密钥管理
  2. 创建新密钥并立即复制保存

⚠️ 密钥只显示一次，之后无法再次查看，务必自行记录。

任务七：撰写公司百科全书

给 AI 提供你的企业资料，后续生成的内容才会专业、不露怯。

7.1 关于你的生意

  • 品牌名、所在国家、主营业务
  • B2B 或 B2C、工厂或贸易商
  • 公司人数、主要出口国家
  • 客户身份、核心竞争优势
  • 推广渠道等

7.2 关于你的客户画像

  • 姓名（虚拟）、国家、年龄、性格
  • 采购产品、对价格 / 质量的敏感度
  • 职位、公司人数
  • 采购偏好国家、生意盈利模式
  • 寻找供应商的渠道、采购的关键关注点
  • 痛点与额外信息
